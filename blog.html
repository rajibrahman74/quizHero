01. What is the difference between map, filter, forEach and find?

➡️ map(): The map method calls a function on each element of the original array and creates a new array, the array that it is called on will stay still. This function is useful for adding minor changes to an already existing array.
syntax: map(function, iterable, ...)

➡️ filter(): filter() is used to create a new array with all elements that pass the given test implemented by the callback function. The original array is not modified. This function returns an array of found results, if any, even if only 1 match is found. If a function finds no match, it simply returns an empty array.
syntax: arr.filter(callback(element[, index[, array]])[, thisArg])

➡️ forEach(): forEach() is used to execute the given function once for each element of the array. It does not return a new array. This function takes 1 parameter. Each(callback) parameter callback can take up to 3 parameters element, index and array (but only element parameter is required, rest is optional).
syntax: arr.forEach(callback(currentValue[, index[, array]])[, thisArg])

➡️ find(): find() is used to return the value of the first element of the array that satisfies the given testing function. It returns undefined if no component tests pass. It does not change the original array. Like the .forEach() function, this function also takes only 1 parameter .find(callback).
syntax: arr.find(callback(element[, index[, array]])[, thisArg])
